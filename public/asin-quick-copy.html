Frequently Asked Questions

Q: How do I install the extension?

A: Download the ZIP file, extract it, then go to Chrome Extensions (chrome://extensions/), enable Developer Mode, and click "Load unpacked" to select the extracted folder.

Q: How to customize keyboard shortcuts?

A: Go to Chrome Extension Settings Page: chrome://extensions/shortcuts and modify the shortcut for ASIN Quick Copy.

Q: Which Amazon sites are supported?

A: The extension works on all major Amazon domains including .com, .co.jp, .co.uk, .de, .fr, .it, .es, .ca, .com.au, and more.

Q: What is an ASIN?

A: ASIN (Amazon Standard Identification Number) is a unique 10-character alphanumeric identifier assigned by Amazon to each product in their catalog.

Q: Does the extension collect any data?

A: No, the extension does not implement data transmission functionality and is completed within the browser.

Q: The extension doesn't work on a specific page. What should I do?

A: Please report the issue on our GitHub Issues page with the URL and we'll investigate.
